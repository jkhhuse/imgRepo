#### 安装webpack
- 本地安装
```shell
> npm install webpack --save-dev
```
官方不推荐使用全局安装的方式，可能会导致使用的webpack与项目中的版本不一致。

#### 创建一个打包示例
- 创建webpack文件夹，并本地安装webpack
```shell
> mkdir webpack-demo && cd webpack-demo
> npm init -y
> npm install webpack --save-dev
```
查看是否成功安装
```shell
> ./node_modules/.bin/webpack --help
```
